- useState 배열에 추가: setVisited([...visited, playerPos]);
```
const currentTile = useMemo(
() => board.find((t) => t.pos.x === playerPos.x && t.pos.y === playerPos.y),
[board, playerPos]
);
```
find는 배열 순환하면서 하나 반환 
include는 있으면 true
some 최소하나있으면 true
	나는 방문한 위치를 배열에 push해뒀다가 playerPos = isVisited 가 있으면 그값을 꺼내려고? 했는데 그냥 배열에 저장, 그리고 some으로 있으면 그냥 true하면 되는 거였음 
	그리고 {}안에 변수 선언이 된다
